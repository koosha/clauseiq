B · Strengths
  • Fastest possible input — palette opens focused.
  • Screen real-estate maximized for clauses, not chrome.
  • Keyboard shortcuts taught inline; power users accelerate.
  • Feels modern, matches Linear / Raycast / Notion mental model.
B · Tradeoffs
  • Discoverability hit for non-keyboard users — chips are less scannable than a sidebar.
  • Drawer adds a mode (open / collapsed) to reason about.
  • Lawyers new to modern tools may feel "empty" at the empty state.
  • Filter count/family counts not visible until you explore.
