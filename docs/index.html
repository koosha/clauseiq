Interactive prototype

PROTO · v0.1Workbench — click-throughlive

Working prototype of the adopted direction. Keyboard-driven: ⌘K opens the command palette, j/k navigate results, p pins, c copies, o opens contract viewer, ⌘↵ exports draft. State persists locally.

try: "cap on liability tied to fees paid in 12 months" → pin 2 clauses → ⌘↵
